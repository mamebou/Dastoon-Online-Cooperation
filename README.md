# Dustoon

複合現実を用いた掃除活性化システム

Paintモード

# Demo

[YouTube](https://youtu.be/msgvNG8r6ms)

# Features

HoloLens2を用いたMRアプリケーション

# Requirement

* Unity 2020.3.18f1 LTS
* Visual Studio 2019
* MRTK 2.7.2

# Usage

Unityでプロジェクトを開く
\[File\] > \[Build Settings...\] > \[Build\]

Visual Studioでslnを開く
HoloLens2をUSBで接続
Release ARM64 Device
\[Debug\] > \[Start Without Debugging\]
